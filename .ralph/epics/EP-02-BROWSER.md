## Goal
Build the Electron browser shell — the core desktop application that users interact with. Tabs, omnibar, sidebar, navigation, and Go engine lifecycle management.

## Acceptance
- Electron app launches with a window showing browser chrome
- Users can open web tabs and browse the internet
- Tab management works (create, close, reorder, switch)
- Omnibar handles URLs, search queries, and commands
- Sidebar shows tab tree with collapsible navigation
- Go engine starts as sidecar and health is monitored
